Como utilizar o endpoint de Professores

Listar Todos os Professores

Para listar todos os professores, faça uma requisição GET para a rota:

/professores

Criar Novo Professor

Faça uma requisição POST para a rota:

/professores

Corpo da requisição (JSON):

{
    "nome": "Nome do Professor",
    "materia": "Disciplina",
    "observacoes": "Observações adicionais"
}

Editar Professor

Faça uma requisição PUT para a rota:

/professores/<id_professor>

Corpo da requisição (JSON):

{
    "nome": "Novo Nome",
    "materia": "Nova Disciplina",
    "observacoes": "Atualizações nas observações"
}

Excluir Professor

Faça uma requisição DELETE para a rota:

/professores/<id_professor>
