Como utilizar o endpoint de Alunos

Listar Todos os Alunos

Faça uma requisição GET para a rota:

/alunos

Criar Novo Aluno

Faça uma requisição POST para a rota:

/alunos

Corpo da requisição (JSON):

{
    "nome": "Nome do Aluno",
    "turma": "Turma",
    "idade": 20,
    "email": "email@example.com",
    "telefone": "123456789"
}

Editar Aluno

Faça uma requisição PUT para a rota:

/alunos/<id_aluno>

Corpo da requisição (JSON):

{
    "nome": "Novo Nome",
    "turma": "Nova Turma",
    "idade": 20,
    "email": "novoemail@example.com",
    "telefone": "123456789"
}

Excluir Aluno

Faça uma requisição DELETE para a rota:

/alunos/<id_aluno>
