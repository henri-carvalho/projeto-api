Bem-vindo ao Projeto de Desenvolvimento de APIs e Microsserviços

Este projeto é desenvolvido para a disciplina de Desenvolvimento de APIs e Microsserviços, sob orientação do professor Caio. A aplicação é um sistema de gerenciamento completo (CRUD) de alunos, professores e turmas, com o armazenamento de dados em um banco de dados SQLite.

Descrição do Projeto

A API RESTful que compõe o projeto permite realizar todas as operações essenciais de um CRUD:

  • Criação: Adicione novos registros para alunos, professores e turmas.
  • Leitura: Consulte detalhes específicos ou listas de alunos, professores e turmas.
  • Atualização: Edite informações de registros existentes.
  • Exclusão: Remova registros de forma segura e eficaz.

O banco de dados SQLite garante um armazenamento leve e de fácil integração, adequado para explorar a prática de manipulação de dados em ambientes de desenvolvimento.

Sobre o Site

Este site é uma interface amigável que facilita o uso e teste das funcionalidades da API. Além disso, oferece documentação para guiar o uso das rotas e orientá-lo nas operações de cada recurso disponível.

Explore o menu de navegação no topo para conhecer cada rota e acesse a aba de documentação para um guia detalhado sobre como realizar chamadas para a API e utilizar cada funcionalidade.
