Como utilizar o endpoint de Turmas

Listar Todas as Turmas

Para listar todas as turmas, faça uma requisição GET para a rota:

/turmas

Criar Nova Turma

Faça uma requisição POST para a rota:

/turmas

Corpo da requisição (JSON):

{
    "descricao": "Descrição da Turma"
}

Editar Turma

Faça uma requisição PUT para a rota:

/turmas/<id_turma>

Corpo da requisição (JSON):

{
    "descricao": "Nova Descrição da Turma"
}

Excluir Turma

Faça uma requisição DELETE para a rota:

/turmas/<id_turma>
