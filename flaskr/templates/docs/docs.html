Documentação da API

1. Listar Alunos

Método: GET

Endpoint: /alunos

Descrição: Retorna uma lista de todos os alunos.

Resposta:

  • 200 OK: Uma lista de objetos de alunos.
[
  {
    "id": 1,
    "nome": "João",
    "idade": 15,
    "turma": "2A",
    "nascimento": "2009-01-15",
    "nota_primeiro_semestre": 8.5,
    "nota_segundo_semestre": 9.0,
    "nota_final": 8.75
  },
  ...
]

2. Obter Aluno por ID

Método: GET

Endpoint: /alunos/<int:aluno_id>

Descrição: Retorna os detalhes de um aluno específico pelo ID.

Parâmetros:

  • aluno_id: O ID do aluno.

Resposta:

  • 200 OK: Objeto do aluno.
  • 404 Not Found: Se o aluno com o ID especificado não for encontrado.
{
  "id": 1,
  "nome": "João",
  "idade": 15,
  "turma": "2A",
  "nascimento": "2009-01-15",
  "nota_primeiro_semestre": 8.5,
  "nota_segundo_semestre": 9.0,
  "nota_final": 8.75
}

3. Adicionar Aluno

Método: POST

Endpoint: /alunos

Descrição: Adiciona um novo aluno.

Corpo da Requisição:

{
  "nome": "João",
  "idade": 15,
  "turma": "2A",
  "nascimento": "2009-01-15",
  "nota_primeiro_semestre": 8.5,
  "nota_segundo_semestre": 9.0,
  "nota_final": 8.75
}

Resposta:

  • 201 Created: O ID do aluno recém-criado.
  • 400 Bad Request: Se os dados fornecidos forem inválidos.

4. Atualizar Aluno

Método: PUT

Endpoint: /alunos/<int:aluno_id>

Descrição: Atualiza os detalhes de um aluno específico pelo ID.

Parâmetros:

  • aluno_id: O ID do aluno a ser atualizado.

Corpo da Requisição:

{
  "nome": "João Silva",
  "idade": 16,
  "turma": "2A",
  "nascimento": "2008-01-15",
  "nota_primeiro_semestre": 9.0,
  "nota_segundo_semestre": 9.5,
  "nota_final": 9.25
}

Resposta:

  • 200 OK: Aluno atualizado com sucesso.
  • 404 Not Found: Se o aluno com o ID especificado não for encontrado.

5. Excluir Aluno

Método: DELETE

Endpoint: /alunos/<int:aluno_id>

Descrição: Exclui um aluno específico pelo ID.

Parâmetros:

  • aluno_id: O ID do aluno a ser excluído.

Resposta:

  • 204 No Content: Aluno excluído com sucesso.
  • 404 Not Found: Se o aluno com o ID especificado não for encontrado.
